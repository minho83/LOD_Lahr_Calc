# 어둠의전설 라르 계산기

어둠의전설 게임의 순수 HP/MP 강화에 필요한 라르를 계산하는 웹 애플리케이션입니다.

## 기능

- HP/MP 강화에 필요한 라르 계산
- 목표 단수 달성에 필요한 HP/MP 계산
- 보유 라르로 올릴 수 있는 HP/MP 역산
- 단수별 풀경험치 소모 라르 계산
- 라르 할인 이벤트 지원 (0~99% 할인)
- 어둠돈 환산 기능 (현금 비용 계산)
- 무료 라르 (일간/주간) 차감 기능
- 목표 기간 설정 기능
- 설정값 로컬 저장 (브라우저 localStorage)
- 반응형 디자인 (모바일/데스크톱 지원)

## 계산 방식

### 경험치 계산

경험치는 HP와 MP 각각의 공식으로 계산됩니다:

- **HP 경험치**: `hp × (hp - 50) × 5`
- **MP 경험치**: `mp × (mp - 25) × 10`

### 고레벨 구간 보정

HP 555만 이상, MP 280만 이상 구간에서는 스탯 증가량이 점점 감소합니다.
이를 반영하여 경험치가 조정됩니다:

```
조정된 경험치 = 기본 경험치 × (기준 증가량 / 실제 증가량)
```

| HP 구간 | 증가량 | MP 구간 | 증가량 |
|---------|--------|---------|--------|
| 0 ~ 555만 | 50 | 0 ~ 280만 | 25 |
| 555만 ~ 560만 | 45 | 280만 ~ 285만 | 20 |
| 560만 ~ 565만 | 40 | 285만 ~ 290만 | 15 |
| 565만 ~ 570만 | 35 | 290만 ~ 295만 | 10 |
| 570만 ~ 575만 | 30 | 295만 ~ 300만 | 5 |
| 575만 ~ 580만 | 25 | 300만 이상 | 1 |
| 580만 ~ 585만 | 20 | | |
| 585만 ~ 590만 | 15 | | |
| 590만 ~ 595만 | 10 | | |
| 595만 ~ 600만 | 5 | | |
| 600만 이상 | 1 | | |

### 풀경험치 라르 계산

풀경험치 소모 시 라르 계산은 실제 게임 메커니즘을 반영합니다:
- HP/MP 1회 업그레이드마다 라르 소모 + 경험치 차감
- 경험치 차감량은 현재 스탯 기준으로 계산
- 고레벨 구간에서는 조정된 경험치 공식 적용

```
├── index.html          # 메인 페이지 (HTML + 인라인 JS)
├── js/
│   └── calculator-core.js  # 경험치 계산 핵심 모듈
├── css/
│   └── styles.css      # 스타일시트
├── images/
│   └── Thumbnail.webp  # 썸네일 이미지
└── privacy.html        # 개인정보처리방침
```

## 사용법

1. 현재 HP/MP를 입력합니다
2. 목표 HP/MP 또는 단수를 입력합니다
3. 설정 버튼에서 할인율, 시세, 무료 라르 등을 조정합니다
4. "계산하기" 버튼을 클릭합니다

## 로컬 실행

```bash
# 간단한 로컬 서버 실행
npx serve . -l 3456

# 브라우저에서 접속
open http://localhost:3456
```
