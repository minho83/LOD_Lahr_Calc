# 어둠의전설 라르 계산기

어둠의전설 게임의 순수 HP/MP 강화에 필요한 라르를 계산하는 웹 애플리케이션입니다.

## 기능

- ✅ HP/MP 강화에 필요한 라르 계산
- ✅ 라르 할인 이벤트 지원 (20%, 30% 할인)
- ✅ 어둠돈 환산 기능 (현금 비용 계산)
- ✅ 체력 30만 이상, 마력 15만 이상부터 라르 소모 계산
- ✅ 반응형 디자인 (모바일/데스크톱 지원)

## 사용법

1. 현재 HP/MP와 목표 HP/MP를 입력합니다
2. 라르 할인율을 선택합니다 (할인 없음 / 20% / 30%)
3. 어둠돈 1억당 현금 가격을 입력합니다
4. "계산하기" 버튼을 클릭합니다
